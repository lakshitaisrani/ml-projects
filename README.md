# Machine Learning Projects

This repository contains a collection of Machine Learning and Deep Learning projects implemented using Python. The projects cover core concepts such as dimensionality reduction, feature selection, clustering, neural networks, and convolutional neural networks on real-world and benchmark datasets.

## Topics Covered
- Principal Component Analysis (PCA)
- Linear Discriminant Analysis (LDA)
- Singular Value Decomposition (SVD)
- Feature Selection Techniques
- K-Means Clustering
- DBSCAN Clustering
- Hierarchical Clustering
- Artificial Neural Networks (ANN)
- Convolutional Neural Networks (CNN)

## Datasets Used
- MNIST Digits Dataset
- Cat and Dog Dataset
- Iris Dataset
- Customer Dataset
- Mobile Price Classification Dataset
- Malaria Dataset
- Aerial Cactus Dataset

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-Learn
- TensorFlow
- Keras

## Projects Included
1. PCA on MNIST Dataset
2. PCA on Cat and Dog Dataset
3. LDA on Cat and Dog Dataset
4. DBSCAN on Iris Dataset
5. SVD on MNIST Dataset
6. Forward and Backward Feature Selection
7. K-Means Clustering on Iris Dataset
8. Hierarchical Clustering on Customer Dataset
9. Neural Network for Mobile Price Classification
10. CNN on MNIST Dataset
11. CNN on Malaria Dataset
12. CNN on Aerial Cactus Dataset
