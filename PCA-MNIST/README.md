# PCA on MNIST Dataset

## Overview
This project demonstrates the application of Principal Component Analysis (PCA) on the MNIST Handwritten Digits Dataset. PCA is used to reduce the dimensionality of the dataset while preserving most of the important information, enabling faster computation and easier visualization.

## Objectives
- Apply PCA for dimensionality reduction.
- Analyze the variance retained by principal components.
- Visualize the transformed data in a lower-dimensional space.
- Understand the impact of PCA on high-dimensional image data.

## Dataset
**MNIST Handwritten Digits Dataset**

- 70,000 grayscale images of handwritten digits (0–9)
- Image size: 28 × 28 pixels
- Commonly used for machine learning and computer vision tasks

**Source:**  
https://keras.io/api/datasets/mnist/
